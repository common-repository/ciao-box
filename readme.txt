== Description ==

Allows the user to display a Ciao Box in the sidebar.

== Installation ==

1. Unzip `ciao_box.zip` and upload the contained files to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

= Add the sidebar widget =

1. Install the `Ciao Box` widget through the 'Design' menu in WordPress
2. Generate a Ciao Box Code at http://www.ciao.de/specials/Ciao_Box_Generator
3. Copy & Paste the generated code in the textarea Ciao Box Code through the 'Design' menu in WordPress

= (Optional, for advanced users) Add `ciao_box` to a post or a page =

1. Type `[ciao_box]` into a post's or a page's body text.

= (Optional, for advanced users) Add `ciao_box` to a template =

1. Enter `<?php echo do_shortcode('[ciao_box]'); ?>` into a suitable template file.
